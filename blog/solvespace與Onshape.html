增加子目錄指令:



A.→git clone 小組倉儲


B.→cd 倉儲資料夾


C.→git submodule add 子目錄倉儲網址 子目錄名稱

更新子目錄指令:



A.git submodule update --init --recursive→抓取原倉儲的子目錄資料


B.git submodule foreach --recursive git pull origin gh-pages→更新所有的子目錄的版本(所有組員都更新)
