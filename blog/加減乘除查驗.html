W10-加減乘除查驗

Date 週四 27 四月 2017 By 40423104-YanSyuan Tags 加減乘除查驗-W10

利用程式做加減乘除的運算

1.先建立一個較m1的資料夾(配合程式)

2.將以下兩個程式碼放在m1資料夾

3.按鈕程式碼

上述按鈕取值程式碼:

加運算

程式碼

檔名append.py

def addend (a,b):
      return a+b

檔名app.py

import sys
sys.path.append("./m3")
import addend


sum = addend.addend (19,88)


print(sum)

減運算

程式碼

檔名minus.py

def minus (a,b):
      return a-b

檔名app.py

import sys
sys.path.append("./m4")
import minus


sum = minus.minus (698,2278)


print(sum)

乘運算

def tines (a,b):

檔名app.py

import sys
