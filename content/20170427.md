###1.先建立一個較m1的資料夾(配合程式)
###2.將以下兩個程式碼放在m1資料夾
##加運算
####程式碼
<p>檔名append.py</p>
<pre>
def addend (a,b):
      return a+b
</pre>
<p>檔名app.py</p>
<pre>
import sys
sys.path.append("./m3")
import addend

sum = addend.addend (19,88)

print(sum)
</pre>
